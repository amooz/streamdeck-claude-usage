# streamdeck-claude-usage

A Stream Deck plugin that displays Claude usage statistics on your Stream Deck buttons.

## Overview

Each button shows a Claude usage metric (for example, tokens used, percentage of quota consumed, or remaining budget). The button's appearance is configurable per-key:

- **Text only** — the metric rendered as text.
- **Circular progress bar only** — the metric rendered as a ring around the key.
- **Text and circular progress bar** — both, combined on the same key.

## Status

Early development. The plugin is not yet packaged for installation.
